基于RaspberryPi的drcom“路由器”

发表于 2018-10-24 | 更新于: 2019-03-18 | 分类于 折腾日记 | |
字数统计: | 阅读时长 ≈

由于校园网的限制，只能一人一号一端口登录校园网，基于GitHub项目： https://github.com/drcoms/drcom-generic 我经过钻研，实现了非openwrt，Pandora等路由器系统，直接通过树莓派实现路由功能。 先分享如何来完成路由器的制作：

一、材料准备

1、硬件

  • raspberrypi 3b/3b+
  • 16/32GB SDcard

2、软件

  • Win32DiskImager
  • SD Formatter
  • Wireshark
  • WinSCP

3、镜像

  • Ubuntu MATE 地址： https://ubuntu-mate.org/raspberry-pi/

二、安装镜像

安装镜像一些过于基础的东西在网上有很多教程了，我就不再赘述了。 附上教程链接：https://blog.csdn.net/kxwinxp/article/details/78370913

三、配置drcom的文件

基于重庆大学d版的教程：d版简略使用和配置说明 GitHub项目的官方教程：

https://github.com/drcoms/drcom-generic/wiki/d%E7%89%88%E7%AE%80%E7%95%A5%E4%BD%BF%E7%94%A8%E5%92%8C%E9%85%8D%E7%BD%AE%E8%AF%B4%E6%98%8E

我觉得这个教程对新手来说十分不友好，所以我对这个过程进行详细过程的说明： 1、使用 wireshark 在官方客户端登录前开始截包，做一次完整的截包动作然后登出，保存为 wireshark截包文件, 比如 dr.pcapng (扩展名为pcapng)。 ①在wireshark中点中自己的连接 ②例如，我的连接是以太网这个名称，所以就选中以太网 ③点击开始捕获分组之后，打开drcom宽带认证客户端然后登陆，注销。 ④停止截包保存 ⑤保存到桌面，命名为drcom.png 进入到GitHub的在线配置网页：http://drcoms.github.io/drcom-generic/ 点击open上传刚才截包文件，获得一串解析的文件。 ⑥开始正式配置latest-wired.py文件 从https://github.com/drcoms/drcom-generic 克隆项目到桌面。 把第五步截包获得的粘贴到 #config 和 #config_end之间 覆盖掉然后保存。记得在password加上自己的宽带密码。 现在我们就完成了drcom文件的配置。

四、连接到树莓派并把latest-wired.py传入树莓派中。

连接到树莓派也是比较基础的操作了，不在赘述。可参考树莓派连接教程。 我是有路由器进行连接的，比较方便。 若没有路由器可参考教程： https://my.oschina.net/quanpower/blog/221920 建议使用VNC viewer进行图形化界面连接，方便操作，由于我是通过显示器连接的，这部分没有相关操作。 把latest-wired.py文件传入树莓派中可以有很多种方法： ①可以通过winscp上传 ②树莓派联网后从邮箱下载 等……….

五、配置Ubuntu mate的热点

进入桌面后，按下面的步骤进行配置。 ssid是热点名称 设置密码wpa&wpa2

六、最后一步，开机自启python脚本

打开终端输入

sudo nano /etc/rc.local

在exit 0 之前加入

python /home/pi/Desktop/latest-wired.py&

七、插网线重启即可，尽情享受吧！
